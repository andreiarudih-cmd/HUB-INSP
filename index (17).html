Plataforma INSP - Código Fonte v1.0

Como Usar este Arquivo

Siga estes passos para rodar a primeira versão da plataforma:

  1. Crie um projeto Next.js: Abra seu terminal e rode o comando:
    npx create-next-app@latest insp-platform --typescript --tailwind --eslint
  2. Instale as dependências: Navegue para a pasta do projeto (cd insp-platform) e rode:
    npm install lucide-react
  3. Crie as pastas e arquivos: Dentro do seu projeto, crie as pastas e arquivos exatamente como os caminhos mostrados abaixo (ex: crie a pasta /lib e dentro dela o arquivo mockData.ts).
  4. Copie e cole o código: Copie o conteúdo de cada bloco de código abaixo e cole no arquivo correspondente que você criou.
  5. Rode o projeto: Após copiar todos os arquivos, rode o comando:
    npm run dev
  6. Abra http://localhost:3000 no seu navegador para ver a plataforma.
